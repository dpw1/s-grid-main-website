Usage with React

If you want to work with React you need to check out this repo: https://github.com/juliancwirko/react-boilerplate. This is a ready to use React project scaffold.

This is a React sample project which uses Webpack and sGrid from the start. After you run it you can go to http://localhost:3000 

$ git clone https://github.com/juliancwirko/react-boilerplate.git
$ cd react-boilerplate
$ npm install
$ npm start
