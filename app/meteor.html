Usage with Meteor

Using it with Meteor projects is even simpler. You just have to install the package.

Then in your project, in your main .styl file, include all three files. The same as in Npm section. You shouldn't install the Stylus package as it will be installed with the sGrid package. And that's it! Really fast and simple.

Just add the package in your project: 

$ meteor add juliancwirko:s-grid
